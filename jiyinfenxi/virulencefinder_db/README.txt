The databases available in this repository are as follows:

For S.aureus:
-s.aureus_exoenzyme.fsa: Exoenzyme genes for S.aureus
-s.aureus_hostimm.fsa: Hostimm genes for S.aureus
-s.aureus_toxin.fsa: Toxin genes for S.aureus

For Escherichia coli:
-virulence_ecoli.fsa: Virulence genes for Escherichia coli
-stx.fsa: Shiga-toxin genes

For Listeria:
-listeria.fsa: Virulence genes for Listeria

For Enterococcus:
-virulence_ent.fsa: Virulence genes for Enterococcus

For Enterococcus faecium & Enterococcus lactis:
-virulence_entfm_entls.fsa: Virulence genes for Enterococcus faecium & Enterococcus lactis
